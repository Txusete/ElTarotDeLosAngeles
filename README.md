# El Tarot de los Ángeles — GitHub Pages

This folder contains the static website for **El Tarot de los Ángeles**, hosted via GitHub Pages.

## Files

| File | Description |
|------|-------------|
| `index.html` | Main landing page |
| `privacy.html` | Privacy Policy (in Spanish) |
| `app-ads.txt` | Google AdSense/AdMob ads.txt declaration |

## Setup

1. Push this folder's contents to the root of a `gh-pages` branch (or to `/docs` on `main`).
2. Enable GitHub Pages in the repository settings → point it to the correct branch/folder.
3. The site will be available at `https://<your-username>.github.io/<repo-name>/`.
